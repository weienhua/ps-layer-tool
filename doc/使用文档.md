# 图层处理工具 - Photoshop 插件使用文档

## 一、安装说明
### 方式一：自动安装（推荐）

下载安装程序，运行即可：

- **Windows**: `com.layertool.panel-installer.exe`，双击运行
- **macOS**: `com.layertool.panel-installer-macos`，右键选择"打开"或使用终端运行：
  ```bash
  chmod +x com.layertool.panel-installer-macos
  ./com.layertool.panel-installer-macos
  ```

安装程序会自动检测 Photoshop 版本、复制插件文件、开启调试模式。安装完成后重启 Photoshop 即可使用。

**卸载方法**：
- **Windows**: 双击运行 `com.layertool.panel-uninstaller.exe`
- **macOS**: 右键选择"打开"或使用终端运行：
  ```bash
  chmod +x com.layertool.panel-uninstaller-macos
  ./com.layertool.panel-uninstaller-macos
  ```

### 方式二：手动安装（zip 包）

下载 `com.layertool.panel-vX.X.X.zip`，解压后将 `com.layertool.panel` 文件夹复制到：

- **Windows**: `%APPDATA%\\Adobe\\CEP\\extensions\\`
- **macOS**: `~/Library/Application Support/Adobe/CEP/extensions/`

然后启用调试模式（见下方"安装步骤"）。


### 安装包内容

```
├── CSXS/manifest.xml          # CEP 扩展清单
├── dist/                      # 面板程序文件
│   ├── index.html
│   ├── style.css
│   ├── bundle.js
│   ├── lib/
│   │   ├── CSInterface.js
│   │   ├── presets.md
│   │   └── template.md
│   └── jsx/hostscript.js
└── doc/使用文档.md
```

### 兼容性

- **Photoshop 版本**: 2019 (v20.0) 及以上
- **系统**: Windows / macOS

### 安装步骤

#### Windows 安装

1. **复制文件夹**

   将整个插件文件夹复制到以下位置：
   ```
   %APPDATA%\Adobe\CEP\extensions\com.layertool.panel
   ```
   
   快速打开该目录：按 `Win + R`，输入 `%APPDATA%\Adobe\CEP\extensions`，回车。

2. **启用调试模式**

   双击运行 `doc/csxs.reg` 目录中对应的注册表文件：
   - PS 2019: `csxs.9.reg`
   - PS 2020-2021: `csxs.10.reg`
   - PS 2022+: `csxs.11.reg`

   也可手动创建注册表项：
   ```
   路径: HKEY_CURRENT_USER\Software\Adobe\CSXS.11
   名称: PlayerDebugMode
   值: 1
   类型: 字符串
   ```

3. **重启 Photoshop**

   在菜单中找到: **窗口 → 扩展功能 → 图层处理工具**

#### macOS 安装

1. **复制文件夹**

   将整个插件文件夹复制到以下位置：
   ```
   ~/Library/Application Support/Adobe/CEP/extensions/com.layertool.panel
   ```
   
   快速打开该目录：Finder → 前往 → 前往文件夹，粘贴上述路径。

2. **启用调试模式**

   打开终端，执行以下命令：
   ```bash
   # PS 2019
   defaults write com.adobe.CSXS.9 PlayerDebugMode 1

   # PS 2020-2021
   defaults write com.adobe.CSXS.10 PlayerDebugMode 1

   # PS 2022+
   defaults write com.adobe.CSXS.11 PlayerDebugMode 1
   ```

3. **重启 Photoshop**

   在菜单中找到: **窗口 → 扩展功能 → 图层处理工具**

## 三、图层信息

### 功能说明

获取选中图层的坐标、尺寸、旋转角度、文字信息等，按模板格式化输出。

### 使用步骤

1. 在 Photoshop 中选中一个或多个图层
2. 选择输出模板（下拉列表或自定义）
3. 选择位置锚点（控制图层坐标计算的参考点）
4. 选择排序方式
5. 点击 **"获取选中图层信息"** 按钮
6. 格式化结果会自动显示并复制到剪贴板

### 模板变量

| 变量 | 说明 | 示例值 |
|------|------|--------|
| `{i}` | 图层序号（从0开始，按排序方式） | `0`, `1`, `2` |
| `{name}` | 图层名（去扩展名） | `icon` |
| `{acname}` | 图层名（去 `_数字`/`拷贝` 后缀） | `time_0 拷贝 2` → `time`，`dot 拷贝 11` → `dot` |
| `{type}` | 图层类型 | `normal` / `smartObject` / `text` |
| `{x}`, `{y}` | 锚点坐标 | `100`, `200` |
| `{width}`, `{height}` | 图层宽高 | `50`, `50` |
| `{centerX}`, `{centerY}` | 中心点坐标 | `125`, `225` |
| `{rotation}` | 旋转角度（度） | `0` |
| `{path}` | 图层组路径 | `groupA/subGroup/` |
| `{text}` | 文字图层内容 | `Hello` |
| `{fontSize}` | 文字图层字号 | `24` |
| `{fontColor}` | 文字颜色 | `#FF0000` |
| `{scaleAnim}` | 缩放动画表达式（自定义） | `(1-0.1*sin(#loop/300))` |
| `{rotateAnim}` | 旋转动画表达式（自定义） | `5*sin(#loop/300)` |

### 数学表达式

模板中支持对**数字类型**的变量进行数学运算。仅以下变量可参与计算：

> `i`, `x`, `y`, `width`, `height`, `rotation`, `centerX`, `centerY`, `fontSize`

支持的运算符：`+` `-` `*` `/` `%`（取模）和括号 `()`。

| 写法 | 说明 | 示例结果 |
|------|------|----------|
| `{i+1}` | 序号从1开始 | `0`→`1`, `1`→`2` |
| `{i*2}` | 序号翻倍 | `0`→`0`, `1`→`2` |
| `{width*2}` | 宽度翻倍 | `50`→`100` |
| `{(i+1)*100}` | 序号+1再×100 | `0`→`100`, `1`→`200` |
| `{x/2+5}` | x 坐标的一半加5 | `100`→`55` |
| `{i%3}` | 序号对3取模 | `0`→`0`, `3`→`0` |

**规则**：
- 只有数字类型的字段可以参与计算，字符串字段（`name`、`path`、`text` 等）不可参与
- 如果表达式中包含非数字字段，该表达式保持原样不替换
- 表达式语法错误时也保持原样不替换

### 支持的函数

模板中支持使用数学函数，函数支持嵌套调用：

| 函数 | 说明 | 示例 |
|------|------|------|
| `round(x)` | 四舍五入取整 | `{round(width/2)}` |
| `round(x, n)` | 保留 n 位小数 | `{round(x/3, 2)}` |
| `ceil(x)` | 向上取整 | `{ceil(height/10)}` |
| `floor(x)` | 向下取整 | `{floor(x/5)}` |
| `int(x)` | 取整数部分 | `{int(rand()*10)}` |
| `abs(x)` | 绝对值 | `{abs(x-100)}` |
| `min(a, b)` | 最小值 | `{min(width, height)}` |
| `max(a, b)` | 最大值 | `{max(x, y)}` |
| `rand()` | 随机数 (0-1) | `{int(rand()*100)}` |
| `pow(x, y)` | x 的 y 次方 | `{pow(2, 10)}` |
| `sqrt(x)` | 平方根 | `{sqrt(width*height)}` |

**嵌套示例**：
- `{int(rand()*10)}` — 生成 0-9 的随机整数
- `{round(pow(2, 0.5), 3)}` — 计算 √2 并保留 3 位小数
- `{max(min(x, 100), 0)}` — 将 x 限制在 0-100 范围内

**注意**：默认所有数值输出保留 2 位小数（整数除外）。

### 锚点选择

3×3 网格选择器，9 个位置：

```
┌─────┬─────┬─────┐
│ 左上 │ 中上 │ 右上 │
├─────┼─────┼─────┤
│ 左中 │ 中心 │ 右中 │
├─────┼─────┼─────┤
│ 左下 │ 中下 │ 右下 │
└─────┴─────┴─────┘
```

选择不同锚点会影响 `{x}` 和 `{y}` 的计算结果。例如选择"左上"，则 x/y 为图层左上角坐标；选择"中心"，则 x/y 为图层中心点坐标。

### 排序方式

- **按 X 升序**: 从左到右排序
- **按 Y 升序**: 从上到下排序
- **按 PS 图层顺序**: 按照图层面板中的顺序

### 预设管理

**统一预设列表**：Tab1（图层信息）和 Tab2（模板输出）的预设统一显示在同一列表中，支持筛选按钮（全部/图层信息/模板输出）。

- **保存预设**: 设置好配置后，输入预设名称，点击"保存预设"
- **应用预设**: 点击预设卡片，自动切换到对应 Tab 并执行获取图层信息
- **删除预设**: 点击预设卡片右上角的 × 按钮
- **拖拽排序**: 拖动预设卡片调整顺序，Tab1 和 Tab2 预设可交叉排序
- **筛选显示**: 点击筛选按钮可按来源筛选预设（全部/图层信息/模板输出）
- **标签区分**: 预设卡片带彩色标签显示来源（蓝色=图层信息，绿色=模板输出）
- **折叠状态**: 折叠状态下显示数字标识（1=图层信息，2=模板输出）

**预设持久化存储**：预设数据统一存储到本地文件和 localStorage，双重备份确保数据不丢失。

| 数据 | 本地文件路径 | localStorage Key |
|------|-------------|------------------|
| 统一预设 | `dist/lib/presets/all/default.json` | `layerTool.presets.all.v1` |
| XML 模板配置 | `dist/lib/presets/tab4/default.json` | `layerTool.xmlConfig.v1` |

安装/卸载脚本会自动备份和恢复 `dist/lib/presets/` 目录，更新插件时预设数据不会丢失。

### 预定义模板

下拉列表中提供多个预定义模板（从 `presets.md` 加载），选择后自动填充到文本域。内置 7 条模板：

| 模板 | 用途 |
|------|------|
| `x="{x}" y="{y}"` | 基础坐标 |
| `x="{x}" y="{y}" w="{width}" h="{height}"` | 坐标 + 尺寸 |
| `<Image src="{path}{name}.png" x="{x}" y="{y}" />` | 简单图片 |
| `<Image src="{path}{name}.png" x="{x}" y="{y}" w="{width}" h="{height}" scaleType="fill"/>` | 图片 + 尺寸 |
| `<Image ... w="{width}{scaleAnim}" h="{height}{scaleAnim}" .../>` | 带缩放动画的图片 |
| `<Image ... rotation="{rotateAnim}" />` | 带旋转动画的图片 |
| `<Text textExp="'{text}'" x="{x}" y="{y}" color="{fontColor}" size="{fontSize}"/>` | 文字图层 |

## 四、模板输出

### 功能说明

使用数组索引语法，精确控制每个图层在输出中的位置。适用于需要批量处理多个图层的场景。

### 使用步骤

1. 在 Photoshop 中选中多个图层
2. 选择或编写输出模板（使用数组索引语法）
3. 选择位置锚点和排序方式
4. 点击 **"获取图层信息"** 按钮
5. 格式化结果会自动显示并复制到剪贴板

### 数组索引语法

使用 `{变量名[索引]}` 格式引用每个图层，索引从 0 开始：

```
<Image src="{path[0]}{name[0]}.png" x="{x[0]}" y="{y[0]}" />
<Image src="{path[1]}{name[1]}.png" x="{x[1]}" y="{y[1]}" />
<Image src="{path[2]}{name[2]}.png" x="{x[2]}" y="{y[2]}" />
```

### 可用变量

| 变量 | 说明 | 示例值 |
|------|------|--------|
| `{name[i]}` | 图层名 | `icon_0` |
| `{acname[i]}` | 图层名（去 `_数字` 后缀） | `icon_0` → `icon` |
| `{type[i]}` | 图层类型 | `normal` / `smartObject` / `text` |
| `{x[i]}`, `{y[i]}` | 锚点坐标 | `100`, `200` |
| `{width[i]}`, `{height[i]}` | 图层宽高 | `50`, `50` |
| `{centerX[i]}`, `{centerY[i]}` | 中心点坐标 | `125`, `225` |
| `{rotation[i]}` | 旋转角度 | `0` |
| `{path[i]}` | 图层组路径 | `groupA/subGroup/` |
| `{text[i]}` | 文字内容 | `Hello` |
| `{fontSize[i]}` | 字体大小 | `24` |
| `{fontColor[i]}` | 字体颜色 | `#FF0000` |
| `{gapX[i]}`, `{gapY[i]}` | 与前一图层的坐标间距（首个为 0） | `10`, `0` |

### 数学表达式

模板中支持对**数字类型**的数组变量进行数学运算。仅以下变量可参与计算：

> `x[i]`, `y[i]`, `width[i]`, `height[i]`, `rotation[i]`, `centerX[i]`, `centerY[i]`, `gapX[i]`, `gapY[i]`, `fontSize[i]`

支持的运算符：`+` `-` `*` `/` `%`（取模）和括号 `()`。

| 写法 | 说明 | 示例结果 |
|------|------|----------|
| `{x[0]+1}` | 第1个图层 x 坐标加1 | `100`→`101` |
| `{width[0]*2}` | 第1个图层宽度翻倍 | `50`→`100` |
| `{(i[0]+1)*100}` | 第1个图层序号+1再×100 | `0`→`100` |
| `{x[0]+x[1]}` | 两个图层的 x 坐标之和 | `100+200`→`300` |
| `{x[1]-x[0]}` | 两个图层的 x 坐标之差 | `200-100`→`100` |

**规则**：
- 只有数字类型的字段可以参与计算，字符串字段（`name[i]`、`path[i]`、`text[i]` 等）不可参与
- 支持跨图层计算，如 `{x[0]+x[1]}`、`{width[0]+width[1]}`
- 如果表达式中包含非数字字段，该表达式保持原样不替换
- 表达式语法错误时也保持原样不替换

### 预定义模板

下拉列表中提供多个预定义模板（从 `template.md` 加载），选择后自动填充到文本域。内置 5 条模板：

| 模板名称 | 用途 | 图层数 |
|----------|------|--------|
| 默认 | 基础多图层输出 | 2 |
| 5位时间srcid写法（08:08） | 时间显示，使用 srcid 属性 | 5 |
| 5位时间srcExp写法（08:08） | 时间显示，使用 srcExp 属性 | 5 |
| 6位日期srcid写法（03/21星期六） | 日期显示，使用 srcid 属性 | 6 |
| 6位日期srcExp写法（03/21星期六） | 日期显示，使用 srcExp 属性 | 6 |

**说明**:
- `srcid` 写法: 使用 `srcid="表达式"` 直接指定图片索引
- `srcExp` 写法: 使用 `srcExp="'路径'+int(变量)+'.png'"` 动态拼接图片路径

## 五、图层处理（导出）

### 功能说明

将图层导出为独立的图片文件，支持多种导出模式和格式。

### 使用步骤

1. 选择导出路径（默认为 PSD 文件所在目录/export）
2. 选择导出格式（PNG / JPG / BMP）
3. 根据需要勾选选项：
   - **导出不可见图层**: 包含隐藏的图层
   - **导出图层信息 XML**: 同时生成 manifest.xml
   - **保留文件夹层级**: 保持 PS 图层组目录结构
4. 选择导出模式：
   - **导出选中图层**: 仅导出当前选中的非组图层
   - **导出选中图层组**: 递归导出选中图层组内的所有子图层
   - **导出全部图层**: 导出文档中所有可见图层
5. 等待导出完成，查看结果列表

### 导出说明

- 导出过程是非破坏性的，不会修改原始文档
- 自动裁剪透明像素
- JPG 格式会自动填充白色背景
- 导出过程中会显示进度条

## 六、XML 模板

### 功能说明

根据图层位置自动生成锁屏主题 XML 代码，支持百分比、温度、步数三种数据类型。

### 使用步骤

1. 选择数据类型：
   - **百分比**: 适用于电量等 0-100 的数值
   - **温度**: 适用于温度显示，支持正负值
   - **步数**: 适用于步数等动态位数的数值
2. 输入变量名（如 `#battery_level`）
3. 在 PS 中按顺序选中对应的数字图层
4. 选择位置锚点和对齐方式
5. 根据需要勾选 **"输出 rotation 属性"**（默认勾选，仅 rotation ≠ 0 时输出）
6. 点击 **"生成 XML 模板"** 按钮
7. XML 代码会自动显示并复制到剪贴板

### 常用变量管理

变量面板显示所有可用的 XML 变量，点击变量名即可填入输入框。

**内置变量**：
- `#battery_level` — 电量
- `#weatherTemp` — 当前温度
- `#dayTempgao2` — 最高温度
- `#nightTempdi2` — 最低温度
- `#humidityNum` — 湿度
- `#steps_value` — 步数
- 更多变量见面板...

**自定义变量**：
- 点击 **"+ 添加变量"** 按钮，弹出自定义弹窗
- 在弹窗中输入变量名（必填，如 `#myVar`）和描述说明（选填）
- 支持 ESC 键或点击遮罩层关闭弹窗
- 变量名为空时会显示错误提示
- 自定义变量会保存到本地文件，下次启动自动加载
- 点击 **"恢复默认"** 会弹出确认弹窗，确认后重置为内置变量列表（保留自定义变量）

### 数据类型说明

#### 百分比
- 最多 3 个图层（百位、十位、个位）
- 变量名默认: `#battery_level`
- 示例: 电量 85% 时，百位显示 0（隐藏），十位显示 8，个位显示 5

#### 温度
- 第 1 个图层为符号位（正/负），后续为数值位
- 变量名默认: `#weatherTemp`
- 至少需要 2 个图层
- 符号位会自动处理对齐

#### 步数
- 1-5 个图层，根据选中图层数自动适配
- 变量名默认: `#steps_value`
- 动态位数：数值不足高位时自动隐藏

### 对齐方式

对齐方式控制数字位数变化时的位移方向：

- **水平**: 左对齐 / 居中 / 右对齐
- **垂直**: 上对齐 / 居中 / 下对齐

## 七、自定义模板

### 图层信息模板（presets.md）

图层信息 Tab 的下拉列表模板来自 `dist/lib/presets.md` 文件。

**文件格式**：

````
```
模板内容1
```
```
模板内容2
```
````

**添加新模板**：

1. 用文本编辑器打开 `dist/lib/presets.md`
2. 在文件末尾添加新的代码块：
   ````
   ```
   你的模板内容，使用 {变量名} 引用图层属性
   ```
   ````
3. 保存文件，重新打开面板即可看到新模板

**示例**：

````
```
<Image src="{path}{name}.png" x="{x}" y="{y}" w="{width}" h="{height}" />
```
````

### 模板输出模板（template.md）

模板输出 Tab 的下拉列表模板来自 `dist/lib/template.md` 文件。

**文件格式**：

````
name:`模板名称`
```
模板内容，使用 {变量名[索引]} 引用图层属性
```
````

**添加新模板**：

1. 用文本编辑器打开 `dist/lib/template.md`
2. 添加新的模板，格式为：
   ````
   name:`你的模板名称`
   ```
   <Image src="{path[0]}{name[0]}.png" x="{x[0]}" y="{y[0]}" />
   <Image src="{path[1]}{name[1]}.png" x="{x[1]}" y="{y[1]}" />
   ```
   ````
3. 保存文件，重新打开面板即可看到新模板

## 八、常见问题

### 面板无法打开或显示白屏

**原因**: CEP 调试模式未启用

**解决**: 
- Windows: 运行 `doc/csxs.reg` 中的注册表文件，然后重启 PS
- macOS: 执行 `defaults write com.adobe.CSXS.11 PlayerDebugMode 1`，然后重启 PS

### "CSInterface is not defined" 错误

**原因**: `dist/lib/CSInterface.js` 文件缺失

**解决**: 确保 `dist/lib/` 目录下存在 `CSInterface.js` 文件

### "EvalScript error" 错误

**原因**: 宿主脚本加载失败

**解决**: 
1. 检查 `dist/jsx/hostscript.js` 文件是否存在
2. 尝试重启 Photoshop

### 修改模板后不生效

**原因**: 面板启动时加载模板，修改后需要重新加载

**解决**: 
- 关闭面板后重新打开
- 或者重启 Photoshop

### 导出失败

**可能原因**:
1. 导出路径不存在或无写入权限
2. 没有选中任何图层
3. 文档未保存（无法确定导出路径）

**解决**:
1. 手动选择一个有效的导出路径
2. 在 PS 中选中至少一个非组图层
3. 先保存 PSD 文件

## 九、快捷操作

- **点击变量提示**: 在模板变量提示区域，点击任意变量名即可复制到剪贴板
- **拖拽排序预设**: 直接拖动预设卡片调整顺序，Tab1 和 Tab2 预设可交叉排序
- **点击预设卡片**: 自动切换到对应 Tab，应用预设配置并获取图层信息
- **预设卡片悬停**: 显示模板内容预览（显示在卡片上方）
- **筛选预设**: 点击筛选按钮可按来源筛选预设（全部/图层信息/模板输出）
