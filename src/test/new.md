## 增加功能
我想增加一个功能:模板输出，类似图层信息模块儿，有预设配置，可以保存预设到预设列表，显示输出结果。标签增加在图层信息和图层处理之间。

预设配置：
1、预设名：设置预设名称。
2、位置锚点：设置获取图层的位置。
3、排序方式：批量获取图层后的信息排列方式。
4、输出模板：可以输入预设的输出模板，只要一个文本域就行。下面展示可用模板可用信息的字段和解释。
5、获取选中图层信息按钮：点击按钮，按照预设配置设置的信息处理图层信息数据。
6、保存预设按钮：点击按钮，将预设配置设置的信息保存到预设列表。
7、获取的信息中新增两个字段，按照排序方式排列后，图层与前一个图层之间的x间距和y间距。比如我选择两个图层，第一个图层的两个数据就是0，第二个图层的数据就是跟第一个图层之间的间距。
8、点击获取按钮以后使用跟图层信息处理模块相同的方法获取选中图层的数据，不同的是输出模版部分，用户可以用到每一个图层的信息字段，去编辑输出模板，可用字段可以使用数组或者你有没有什么好的方法，下面我给你一个例子：
选中两个图层，点击获取信息，这时我的输出模板是：
```
<Image src="{path[0]}{name[0]}.png" x="{x[0]}" y="{y[0]}" />
<Image src="{path[1]}{name[1]}.png" x="{x[1]}" y="{y[1]}" />
```
输出的数据是：
```
<Image src="1/day/wd/wd/sz_0.png" x="746" y="1431" />
<Image src="1/day/wd/wd/sz_0.png" x="767" y="1435" />
```
预设列表：
功能跟图层信息模块预设列表相同。
输出结果：
功能跟图层信息模块预设列表相同。
